#Sales Performance Dashboard (Excel Project)

#Overview
This project is an interactive sales dashboard built using Microsoft Excel to analyze business performance.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Slicers

## 📊 Features
- Data cleaning and transformation
- Sales analysis by Region, Category, and Product
- Monthly sales trend visualization
- Order status tracking
- Interactive filters (Slicers)

## 📈 Key Insights
- Apparel contributes the highest sales (~50%)
- East & South regions generate maximum revenue
- Significant drop in sales observed in April
- Orders are evenly distributed across statuses
